学习如何使用Hexo写博客

发表于 2019-12-11

我是通过跟着B站视频做出来hexo的，链接：https://www.bilibili.com/video/av44544186

1.下载Node js 和git

2.安装hexo并使用

1).安装hexo

在cmd中输入下面代码 

//利用npm安装cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org

//利用cnpm安装hexo
cnpm install -g hexo-cli

2).新建文件夹blog

路径名： c:\Users\Administrator\blog

3).需要有git的文件配置

安装git后在电脑中有git bash here这个文件，类似于cmd的命令窗口，打开，输入下面代码，用户名和邮箱要写和github的用户名邮箱一样。

git config --global user.name "youname"
git config --global user.email "aa@qq.com"

这时候在c:\Users\Administrator\下有一个名字叫.gitconfig文件，可以打开看到里面的内容

4).初始化hexo

安装git后电脑中有git CMD，打开并输入下面代码 

hexo init

5).启动hexo

在c:\Users\Administrator\blog目录下，用cmd执行下面的代码，启动hexo

hexo s

执行完后可以在浏览器输入localhost:4000来访问

6).写文章

我用的是vscode写文章，文件类型为markdown类型，扩展名是md

将文章放到c:\Users\Administrator\blog\source_posts文件夹下

在c:\Users\Administrator\blog目录下执行下面代码，清理并生成hexo文件

//清理
hexo clean

//生成
hexo g

刷新一下浏览器localhost:4000，现在本地就可以看到文章了

3.部署到github上

1).新建github仓库

在github上要新建github仓库，名称格式必须是 github的昵称.github.io （我的名称 : Luyame.github.io）

2).安装并把文件部署到远端

1.安装，在cmd执行如下代码

cnpm install save --hexo-deployer-git

打开C:\Users\Administrator\blog下的_config.yml文件，如下修改

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: 'git'
//这里的repo改成你自己github上的仓库地址
repo: https://github.com/Luyame/Luyame.github.io.git
branch: master

2.部署到远端，在C:\Users\Administrator\blog目录下，cmd中输入下面的代码:

hexo d

输入之后会要你输入账号和密码，填写正确后就部署到github仓库中了，之后就可通过浏览器访问github仓库名 Luyame.github.io
