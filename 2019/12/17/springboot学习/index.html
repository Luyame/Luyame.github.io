springboot学习

发表于 2019-12-17

学习springboot

1.导入springboot文件报错ServletInitializer 类

The type org.springframework.web.WebApplicationInit

解决：将项目删除，重新导入到MyEclipse
 

修改pom.xml文件将版本改为2.1.3

<version>2.1.3.RELEASE</version>

 

2.在pom.xml文件中报错

Description Resource Path Location Type
Cannot detect Web Project version. Please specify version of Web Project through Maven project property <webVersion>.
E.g.: <properties> <webVersion>3.0</webVersion> </properties>

解决：在<properties>标签中添加<webVersion>3.0</webVersion>

<properties>
    <java.version>1.8</java.version>
    <webVersion>3.0</webVersion>
</properties>

 

3.测试包报错

jupiter包导入有误，如图01

The import org.junit.jupiter cannot be resolved

图01

解决：使用junit

 

4.单元测试的返回值类型必须是void

不然就会报下面的错误
initializationError(org.junit.runner.manipulation.Filter)
