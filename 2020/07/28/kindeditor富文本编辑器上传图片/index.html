kindeditor富文本编辑器上传图片

发表于 2020-07-28

官方文档 http://kindeditor.net/docs/upload.html

一、引用js、css

在页面中引用kindeditor的js和css

<link rel="stylesheet" href="../plug-in/kindeditor/themes/default/default.css" />
<script charset="utf-8" src="../plug-in/kindeditor/kindeditor-all-min.js"></script>
<script charset="utf-8" src="../plug-in/kindeditor/lang/zh-CN.js"></script>

二、页面添加textarea

这里的文本域id是自定义的

<textarea id="about"></textarea>

三、在js中创建对象

//构建一个默认的编辑器
setTimeout(kedit, 500);

function kedit() {
    keditor = KindEditor.create(    
        '#about',    //这里的id要和页面的文本域id一致
    {
        width : "500px",    
        height : "300px",    
        resizeMode : 2,     
        allowUpload : true,
        uploadJson : base_manage+'saveKindEditorImage',    //上传图片接口
        afterBlur : function() {
            $("#about").val(this.sync().html());    //将图片回显到文本域中
            about_content=this.sync().html();   //about_content是我定义的全局变量
            console.log(this.sync().html());
        }
    });    
    keditor.html(about_content); //这个用于初始化数据，将数据显示到文本域中（不需要初始化数据这行代码可以不写）
}

四、上传图片后端接口

请求参数：

POST参数
imgFile: 文件form名称   //必传
dir: 上传类型，分别为image、flash、media、file  //必传

返回格式：

//成功时
{
        "error" : 0,
        "url" : "http://www.example.com/path/to/file.ext"
}
//失败时
{
        "error" : 1,
        "message" : "错误信息"
}

后台接口：

@ResponseBody
@RequestMapping("saveKindEditorImage")  //上传图片接口
public String saveKindEditorImage(HttpServletRequest request,@RequestParam("imgFile") MultipartFile imgFile,@RequestParam("dir") String dir) throws Exception {
    String path = request.getSession().getServletContext().getRealPath("resource/images/editor/");
    String src=centerservice.uploadImage(path, imgFile);
    String result = "{\"error\":0,\"url\":" + JsonUtil.objectToJson("../resource/images/editor/"+src) + "}";
    return result;

}
